# Netflix_SQL_Project

Netflix Data Analysis using SQL
Project Overview

This project analyzes a Netflix dataset using SQL queries to extract meaningful insights about movies, TV shows, actors, genres, directors, and release trends.

The goal is to practice real-world data analysis skills using structured queries on a large dataset.

🎯 Objectives
Analyze Netflix content distribution (Movies vs TV Shows)
Identify most popular genres and actors
Study content trends over time
Perform data cleaning using SQL techniques
Practice advanced SQL (joins, aggregation, string functions)
🛠️ Tech Stack
Database: MySQL
Language: SQL
Tools: MySQL Workbench /
Dataset: Netflix Titles Dataset (Kaggle or similar source)



-- Dataset Features --

show_id – Unique ID of each title
type – Movie / TV Show
title – Name of the content
director – Director name
cast – Actors in the show
country – Country of production
release_year – Year of release
rating – Content rating
duration – Duration (minutes / seasons)
listed_in – Genre categories
description – Summary of content
